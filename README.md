# D-Stage NFT

The frontend(client side) of a NFT selling application 
## Run Locally

Clone the project

```bash
  git clone https://github.com/Bitee-cd/d-stage_updated.git
```

Go to the project directory

```bash
  cd d-stage_updated
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
